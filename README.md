## 📌 Project Overview

This project builds a regression-based machine learning pipeline to estimate house prices based on features like location, area, number of rooms, and more. The goal is to help buyers, sellers, and analysts get data-driven property valuations.

---

## ✨ Features

- 📊 Exploratory Data Analysis (EDA) with visualizations
- 🧹 Data preprocessing — handling missing values, outliers, and encoding
- 🤖 Multiple regression models compared (Linear, Ridge, Lasso, Random Forest)
- 📈 Model evaluation with RMSE, MAE, and R² metrics
- 🔍 Feature importance analysis
- 📓 Clean, well-documented Jupyter Notebook

## 🚀 Getting Started

### Prerequisites

```bash
Python 3.8+
pip
```

### Installation

```bash
# Clone the repository
git clone https://github.com/apoorvai-ai/house-price-estimator.git
cd house-price-estimator

# Install dependencies
pip install numpy pandas scikit-learn matplotlib seaborn jupyter

# Launch the notebook
jupyter notebook housing_price_estimator.ipynb
```
